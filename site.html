CSS

O CSS, que significa Cascading Style Sheets, é a parte que cuida do visual do site. Depois que eu monto a estrutura com HTML, eu uso o CSS para deixar tudo mais bonito e agradável para o usuário. Com ele, eu posso mudar cores, fontes, tamanhos, espaçamentos, alinhar elementos, criar animações e muito mais. Ele também me ajuda a deixar o site responsivo, ou seja, funcionando bem tanto em celulares quanto em computadores. O CSS pode ser aplicado direto nas tags do HTML, mas o mais comum é usar um arquivo separado só pra ele, o que deixa o código mais limpo e organizado.

HTML5

HTML, que significa HyperText Markup Language, é a linguagem que eu uso para criar a estrutura de uma página na web. É com ele que eu defino tudo o que vai aparecer na tela: títulos, parágrafos, imagens, links, botões, listas, tabelas, formulários, entre outros elementos. Cada parte da página é representada por uma tag, como h1 para título principal, p para parágrafos, img para imagens e assim por diante.Uma das vantagens do HTML é que ele é fácil de aprender e bem organizado. Ele também é muito usado junto com outras tecnologias, como o CSS e o JavaScript.

Java Script

O JavaScript é a linguagem de programação que deixa o site interativo e dinâmico. Com ele, eu posso criar funcionalidades que reagem ao que o usuário faz, como clicar em botões, preencher formulários, abrir e fechar menus, mudar conteúdos sem recarregar a página, mostrar mensagens, entre várias outras coisas. Por exemplo, quando você entra num site e tem uma animação rolando, ou um aviso aparece na tela depois que você preenche algo errado — tudo isso é feito com JavaScript.

Java

Java é uma linguagem de programação muito popular e consolidada no mercado. Diferente do JavaScript, que é mais voltado para o desenvolvimento web e roda no navegador, o Java é usado para criar aplicações completas, que rodam em servidores, computadores, celulares e até dispositivos embarcados. Ele é muito usado em empresas, especialmente em sistemas bancários, aplicações corporativas, aplicativos Android e até jogos.

Bootstrap

Bootstrap é um dos frameworks que mais uso quando quero criar um site bonito, moderno e responsivo de forma rápida. Ele é um conjunto de ferramentas pronto, feito em HTML, CSS e JavaScript, que já vem com vários componentes e estilos prontos para usar. Isso economiza muito tempo no desenvolvimento, principalmente quando quero montar uma estrutura visual sem ter que escrever tudo do zero.

Angular

Angular é um framework que eu uso (ou estou aprendendo a usar) para desenvolver aplicações web mais complexas e completas. Diferente do HTML, CSS e JavaScript puros, o Angular já vem com uma estrutura pronta que ajuda muito na organização do código e na criação de projetos grandes.O que mais me chama atenção no Java é que ele é uma linguagem orientada a objetos, o que significa que tudo é baseado em classes e objetos. Isso ajuda a organizar melhor o código, tornar os projetos mais reutilizáveis e facilitar a manutenção a longo prazo.
